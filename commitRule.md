# Commit-rule

### Commit Type

> 커밋 타입

| Commit Type | Description      |
| ----------- | ---------------- |
| 📑 ::        | 파일 생성        |
| ⚡️ ::        | 기능 업뎃        |
| 🪓 ::        | 기능 & 위젯 삭제 |
| 🐛 ::        | 버그 수정        |
| ⚙️ ::        | 프로젝트 세팅    |
| ♻️ ::        | 코드 리팩토링    |
| 🔀 ::        | PR               |
| 🤏 ::        | 미미한 부분 수정 |

ex)  ⚙️ ::  http 라이브러리 추가
